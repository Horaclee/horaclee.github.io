Jeu d'Echec

  • Tech Stack: C++, SFML
  • Github URL: Project Link

L’objectif de ce projet était de développer un jeu d’échecs en C++ intégrant une intelligence artificielle basée sur un algorithme de planification pour jouer contre un adversaire humain ou une autre IA. L’IA devait être capable de prendre des décisions stratégiques en anticipant plusieurs coups à l’avance.


Architecture du projet

  • Une matrice 8x8 stockant les pièces.
  • Des méthodes pour placer, déplacer et supprimer des pièces.
  • Une gestion des règles du jeu.

    • Représentation des pièces

    Chaque pièce (roi, reine, tour, fou, cavalier, pion) est représentée par une classe dérivée de Piece, qui implémente :

  • Son type.
  • Ses mouvements valides.
  • Une méthode peutSeDeplacer(Position dest, Echiquier &e) pour vérifier la validité d’un déplacement.

    • Gestion du jeu

  • Alternance des joueurs.
  • Détection d’échec et mat.

    • Intelligence Artificielle : Planification des coups

    Minimax

  • Il évalue chaque position selon une fonction d’évaluation
  • Il simule les coups adverses pour anticiper les meilleures réponses.

    • Élagage Alpha-Bêta

  • Table de transposition : Mémorisation des positions déjà analysées pour éviter des calculs redondants.
  • Deep Learning / Réseau de neurones : Pour un IA plus avancée
